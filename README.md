# image-upload
  
Features

  * Multiple file upload
  * Preview
  * No browser plugins required
  * No library dependencies

Supported browsers:

  * IE 10+
  * Firefox 4+
  * Chrome 7+
  * Safari 6+
  * Opera 12.1+
  * iOS Safari 6.1+
  * Android 3+
  * IE mobile 10+
